## Demo URL

<https://sample-calculator-app-v2.netlify.app/>

## Setup

1. Install NodeJS version 14 or higher
1. Install dependencies
    ```
    npm install
    ```

## Run

1. Start the app at `http://localhost:3000`
    ```
    npm start
    ```
